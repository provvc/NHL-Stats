NHL Statistics Application
-
An NHL statistic application that will function as a typical skater (player/goalie), team, and game statistic tracker with in-depth analytics. As well, it will feature a trade-value system that will, as accurately as possible, estimate the market value (not $$) of skaters and draft picks. This will help to determine the fairness of skater and draft pick transactions, as well as removing the subjectivity of skater contract salaries (GM, skater agent, and team salary cap management implications) to determine an accurate and object skater rating/score.

Current Stage:
- Creating the ERD (Entity Relationship Diagram) for the system
- Applying the ERD structure to the project folder
- Navigating NHL API to understand which endpoints the data will be pulled from and seeded into this system's database.

Next Stage:
- Create project RESTful API that consumes the NHL API; this API will then seed data into this system's database
- Create secondary project API that will CRUD the system's database once it has been seeded
- API Security
